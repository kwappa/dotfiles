# dotfiles
My dotfiles.

## dotfiles

dotiles below `config`

### how to install

```
% ./setup.sh install
```
* create symlinks from here to your home directory

### how to uninstall

```
% ./setup.sh uninstall
```
* remove symlinks from your home directory

## brewfile.sh

install something via Homebrew.

execute `./brewfile.sh` after installation of Homebrew.
