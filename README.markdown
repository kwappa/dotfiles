# dotfiles
my dotfiles.

## how to install

```
% ./setup.sh install
```

* create symlinks from here to your home directory
* download icons for autotest

## how to uninstall

```
% ./setup.sh uninstall
```

* remove symlinks from your home directory

## dotfiles

* .zshrc
* .gitconfig
* .gitignore
* .rspec
* .vimrc
* .tmux.conf
* .inputrc
* .pryrc
